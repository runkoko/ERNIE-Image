# 基于ERNIE-Image的漫画风分镜头生成器 
现在不管是追剧、看动漫还是自己脑洞小故事，很多人都想把脑海里的剧情画成分镜头，但大部分人不会画画，纯手绘又麻烦又费时间。 借助百度飞桨文心大模型 ERNIE-Image 强大的图像生成能力，做一个简单好玩的漫画风分镜头生成器，只需要输入文字描述，就能自动生成对应风格的分镜图，让普通人也能轻松把脑洞变成可视化的漫画分镜，主打趣味创作和日常娱乐使用。

项目目标
基于百度飞桨 ERNIE-Image 模型，实现文字自动生成漫画风分镜头图片的核心功能，支持简单剧情、场景、角色描述输入，生成风格统一、构图清晰的漫画分镜。
优化生成效果，让画面更贴合漫画分镜排版，提升整体观感和趣味性。
打造轻量、易用的娱乐向工具，降低普通人创作分镜的门槛，实现稳定可用的演示效果，满足日常脑洞创作、娱乐分享需求。
